# Получение матрицы зависимостей

## CLI

### Основное применение

Разбито на подкоманды:

#### matrix

Строим матрицу зависимостей от версии языка и либ

Логика следующая:

1) если флаг `--with-eol` не передан,  то проверяем версию на `https://endoflife.date`
2) в pypi|npm по апи проверяем:
  1) версии рантайма, например в pypi можно проверить поле `required_python`
  2) Версии других зависимостей (если переданы другие либы, например `django-debug-toolbar`) (там может быть N+1 проблема, надо подумать потом. пока будем считать что мы проверяем только рантайм и одну либу)

По умолчанию берем только актуальные версии

`[script] matrix <lang> -l <library>`

- `<lang>` -- Имя рантайма, Например python, node
- `-l|--lib` - либа, может быть несколько, либо через, либо повтор аргумента
- `--with-eol` - включает в себя версии которые уже закончили свою поддержку. Актуально для поиска старых комбинаций либ

рантайм и либа может быть включать версию, например ">=4.11"
